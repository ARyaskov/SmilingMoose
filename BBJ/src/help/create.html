Создание диаграммы
    Меню:

        

        

        

        

        

        

        

        
Для создания объекта диаграммы необходимо:
  
   * для добавления комментария, выбрать его на панели и перетащить на желаемое место на сцене;

   * для добавления линии жизни, выбрать ее на панели и перетащить на желаемое место на сцене;

   * для добавления исполнителя, выбрать его на панели и перетащить на желаемое место на сцене;

   * для добавления сообщения, выбрать источник, выбрать приемник и щелкнуть по изображению соответствующего сообщения.
