Изменение диаграммы
    Меню:

        

        

        

        

        

        

        

        
  В качестве изменеия диаграммы пользователь может:
   
     1.) Перемещеать объекты по сцене.
     2.) Изменять текст в объектах по двойному щелчку мыши.
     3.) Изменять размер линий жизни.
